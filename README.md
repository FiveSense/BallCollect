# Small Game of Ball（小球收集游戏）
**简介**：这是一款基于九轴传感器和次元台的球球收集小游戏，面向小朋友和娱乐场所。主要形式仿照黄金矿工和愤怒的小鸟，击飞弹性物体获得分数，关卡制的休闲小游戏。
